## 📦 Deploying BILAL-MD

**Step 1:** Fork This Repository  
Click the "Fork Repo" button above or use your GitHub dashboard.

**Step 2:** Generate Pairing Code  
Link your WhatsApp account using the Telegram Bot or Web Pairing interface.

> After generating `creds.json`, it will be saved in the `IGRIS-XD/pairing/` folder.

## ⚙️ Features

- ✅ Group Tag All (`.tagall`)
- 🛡️ Admin-Only Restrictions
- 🎮 Mini Games (Tic Tac Toe, etc.)
- 🔈 Text-to-Speech
- 🖼️ Sticker Maker
- 🔗 Anti-Link Detection
- 🚫 Direct Remove System (No Warnings)
- 🎧 Music / YouTube Download

## 📖 About BILAL-MD

BILAL-MD is a Node.js-based WhatsApp bot powered by Baileys. It automates group tasks, entertains users, and provides advanced utilities in WhatsApp. It supports **multi-device** and can be hosted on Heroku, Pterodactyl, and more.
